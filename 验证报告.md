## 使用建议

### 首次使用
1. 修改 `config/device_config.py` - 设置实际设备序列号
2. 修改 `config/app_config.py` - 设置实际包名和Activity
3. 修改 `pages/*.py` - 根据实际应用调整元素定位器
4. 运行测试: `pytest`

### 生成报告
```bash
# 运行测试（自动生成 allure-results）
pytest --alluredir=reports/allure-results

# 生成HTML报告（需安装allure）
allure generate reports/allure-results -o reports/allure-report
allure open reports/allure-report
```

### 注意事项
1. **必须使用项目虚拟环境的 Python**: `D:\测试资料\project\test\Scripts\python.exe`
2. **元素定位器是占位符**: 必须根据实际应用修改，否则测试会失败
3. **需要设备连接**: 模拟器或真机，需确保 adb devices 能看到设备
