# 🚀 快速开始指南 (小白专用)

## 第一步：准备工作
1. 确保电脑已安装 Python 3.8+
2. 手机开启 USB 调试模式
3. 用 USB 线连接手机和电脑

## 第二步：一键安装
```bash
# 1. 打开命令行，进入项目目录
cd android_auto_test

# 2. 安装所有依赖
pip install -r requirements.txt

# 3. 安装 Allure (生成漂亮报告)
python install_allure.py
```

## 第三步：配置设备
1. 打开 `config/config.yaml` 文件
2. 找到 `serial: "0123456789ABCDEF"`
3. 替换为你的设备序列号 (运行 `adb devices` 查看)

## 第四步：运行测试
```bash
# 最简单的方式 - 运行全部测试
python run.py

# 或运行特定模块
python run.py --module settings
```

## 第五步：查看报告
测试完成后：
1. 打开 `reports/allure-report/index.html` (推荐)
2. 或打开 `reports/report.html`

## 🆘 遇到问题？
1. **设备连接失败** → 检查 USB 调试是否开启
2. **Allure 未安装** → 运行 `python install_allure.py`
3. **依赖安装失败** → 运行 `pip install -r requirements.txt`
4. **其他问题** → 查看 `logs/` 目录下的日志文件

## 📞 快速帮助
```bash
# 查看所有可用命令
python run.py -h

# 查看所有测试用例
python run.py --list
```

## 🎯 推荐流程
1. 第一次使用: `python run.py --list` (看看有什么测试)
2. 体验测试: `python run.py --module settings` (运行设置测试)
3. 查看报告: 打开 `reports/allure-report/index.html`

---
**💡 提示**: 所有操作都在项目目录下进行，确保你在 `android_auto_test` 文件夹中！
